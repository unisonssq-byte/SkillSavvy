Создай современный и интерактивный сайт-портфолио "Unicode's Portfolio" для фрилансера-программиста.

Основные требования:

Структура сайта:

Главная страница с блоком текста/медиа, куда попадает пользователь.

Возможность создавать новые страницы. Список страниц находится сверху сайта, по центру, растягивается влево и вправо при добавлении страниц.

Возможность изменять название страницы и добавлять новые через кнопку с плюсом рядом с названием.

Каждая страница состоит из блоков:

Текстовые блоки с фоном чуть светлее основного фона и закругленными краями.

К каждому текстовому блоку можно добавлять изображения/видео справа от блока. Кнопка “+” для добавления медиа следует за мышью по правому краю блока.

Блоки могут быть неограниченно добавлены, под текстом автоматически расширяется фон блока.

Каждое изображение/видео привязано к конкретному блоку.

Редактирование блоков: кнопка с ручкой в левом верхнем углу блока, доступна только для администратора.

В режиме редактирования блок имеет кнопки подтверждения и отказа (справа внизу). Отказ — с предупреждением и подтверждением удаления блока.

При добавлении нового блока текст/медиа автоматически интегрируется в блоки страницы.

Администраторский доступ:

В правом верхнем углу сайта — закругленная кнопка с иконкой админа.

При нажатии появляется окно для ввода пароля (берется из password.py).

Если пароль правильный — сохраняются права администратора на данном и других ПК, даже после закрытия вкладки.

Если пароль неверный — закрыть окно и показать уведомление “Неправильный пароль”.

Администратор может редактировать/добавлять блоки на всех страницах.

Дизайн:

Темная тема: темно-фиолетовый, темно-синий, черный, серый, светлый текст (не чисто белый).

Все элементы кроме текста: закругленные, полупрозрачные, плавные анимации.

Огромное количество анимаций: плавное открытие блоков, добавление медиа, переход между страницами.

Контент:

Возможность вставки текста, фото, видео и GIF.

Вставка медиа расширяет блок по ширине, текстовый блок остается на месте.

Возможность создавать блоки в любом месте страницы через мышь (не только сверху/середины блока).

Техническая реализация:

Сайт работает как мини-браузер без адресной строки.

Все изменения синхронизируются у всех пользователей в реальном времени (например, через backend на Node.js или Python + WebSocket).

Хранение данных на сервере (текст, изображения, видео, страницы).

Файлы и страницы можно редактировать и сохранять администратором.

Инструкции по установке:

Подробная инструкция по загрузке сайта на Linux-хостинг через консоль и через ISPmanager.

Все команды для Linux (создание папок, права, перенос файлов).

Пошаговое объяснение, куда размещать файлы, как запускать сервер, как подключить домен unicodeportfolio.ru.

Выход:

Полностью готовый проект сайта с фронтендом, бэкендом, базой данных (или JSON-файлами для хранения контента), скриптом админ-пароля.

Все страницы, блоки, кнопки и анимации должны быть интерактивными.

Подробная инструкция по установке и настройке на Linux-хостинг с сохранением прав администратора.
